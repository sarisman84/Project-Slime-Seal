--------------------
Cartoon Skyboxes
--------------------

This package contains 8 high-resolution skyboxes for Cartoon/Comic games. There are 3 types of sky (day, night and sunrise/sunset) in this package. Each skybox has two styles for you to choose to match the style of your game.

Texture size:
8192 x 4096 pixels

Package comes with sample script and demo scenes that help you quickly preview the skyboxes. The demo scenes are located in the folder "\Assets\Cartoon Comic Skyboxes\Demo Scenes\".

This project has been tested on unity 2017.1.0/ 2018.4.3/ 2019.2.12.
